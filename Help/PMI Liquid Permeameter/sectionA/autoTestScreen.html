Auto Test Screen

The Auto Test Screen displays the data acquisition in real time in a graph and also displays the current state of the machine.




Tank Levels
When first run, the program will first check the level of liquid in the tanks. There are two tanks: the collected volume tank and the reservoir tank. The liquid comes from the reservoir and is pumped through the sample and ends up in the collected volume tank. If the volume in the collected volume tank is above a certain level the program will flush the tank. This may take a while. The level at which this flushing is triggered can be set in Settings>Machine Settings>"Close flushing valve at collected volume of:" as a percentage of the total volume of the tank. Make sure that the reservoir tank is full before and during the test. You can add more liquid to the reservoir tank during the test. Be aware that the program will keep on checking the tank levels during the test and will stop the test if the tank level is full and flush it automatically, at which point you should add more liquid to the reservoir tank.





Target Temperature and Target Pressure
The program first attempts to reach the target temperature of the first step in the program. Only the currently selected chamber will be heated, as well as the reservoir tank. Note that it might take a long time to reach the target temperature. As soon as the target temperature is reached the program will start the main pump to build up pressure until it reaches the target pressure of the current step. The program will attempt to maintain a constant pressure during the course of the step in the test by automatically adjusting the speed of the pump. Note that the program is better at maintaining constant pressure at lower pressures than at high pressures. Also note that the program is designed to work best with steps of increasing temperature and increasing pressure (rather than decreasing values in each step).





Overlay graphs
Below the graph are 3 checkboxes that allow you to display overlaying graphs of the pressure, temperature and permeability of the current test.



Stop Test Button
When clicking the stop test button the test will finish and a new report window will be displayed. Please note that it might take a while until the test stops (usually until the next data point is acquired). When the test is finished the fans will turn on to cool down the chambers. These fans will turn off automatically when a new test is started. They can also be turned off from the Manual Control window.
