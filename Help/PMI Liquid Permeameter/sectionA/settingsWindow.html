Settings Window

The settings window contains important settings for the program.


COM Port
The first setting is the COM port. It is crucial that we have the right COM port selected or else the machine will not work. Our COM port will only be displayed when we connect the machine to the computer via USB. Also the machine has to be turned on (with the switch at the back of the machine) for the COM port to be displayed. You might see other COM ports of other devices that are connected to the computer which you must not select. You can figure out which COM port to choose by plugging and unplugging the USB from the computer and refreshing the list, the new COM port that appears when you plug in the USB corresponds to your machine.

Data acquisition interval
This is the interval in seconds between each data point that will be recorded and displayed. We recommend an interval of 30 seconds to smooth out irregularities.

Pressure Tolerance
A pressure tolerance of 0.1 PSI means that the pump will start to adjust the pressure when it is 0.1 PSI or more below the target pressure. A lower tolerance could lead to overshooting of the target pressure so a value of 0.1 is recommended.

Temperature Tolerance
Same as the pressure tolerance but for temperature. Temperature varies more slowly than pressure so a value of 3 is recommended. The test will not start until the temperature is within the tolerance value.


Machine Settings



In the machine settings we find the Max Gauge Pressure which is the maximum pressure that the pressure gauge allows. This should never be changed unless the Pressure Gauge is replaced. The other setting is "Close flushing valve at collected volume of:". This is used by the machine when the collected volume tank is full and is flushed automatically. If this value is 13, the valve that drains the collected volume tank will close when the volume drops to 13% of the tank volume. This value can be changed if we want more or less volume to be left in the collected volume tank after the draining procedure. Note that there is a delay between this value being reached and the valve closing completely which means that the volume will end up being lower than that.
